Monopoly: Rannasee Edition [Regeln]

Startgeld für die Spieler:

  • Spieler 1 erhält 1500 Einheiten.
  • Spieler 2 erhält 1500 Einheiten.
  • Spieler 3 erhält 1500 Einheiten.
  • Spieler 4 erhält 1500 Einheiten.

Lustige Regeln:

  1. Bei jedem Pasch müssen die Spieler einen witzigen Tanz aufführen, ansonsten zahlen sie eine Strafe.
  2. Wenn ein Spieler auf "Frei Parken" landet, kann er eine Minute lang verrückte Geräusche machen und Geld von der Bank erhalten.
  3. Die "Gefängnisstrafe" wird aufgehoben, wenn der Spieler einen Witz erzählt, den alle anderen lustig finden.
  4. Jeder Spieler darf einmal pro Spiel eine "Monopoly-Melodie" summen. Die anderen Spieler müssen raten, um welches Lied es sich handelt. Richtig geratene Lieder bringen Belohnungen.
